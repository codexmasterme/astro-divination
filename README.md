# 星语塔罗 · 星象塔罗占卜

以本命星盘为引，借塔罗牌启示前路的纯静态占卜 Web App。零依赖，直接用浏览器打开即可运行。

## 占卜原理

西方占星认为出生瞬间日、月、行星在黄道十二星座与十二宫的分布，构成个人的「本命盘」，象征性格底色与人生课题。其中**太阳星座**是核心自我（由出生日期定），**上升星座**是你面向世界的姿态（需出生时间＋地点，结合地方恒星时推算）。塔罗牌则以完整 78 张（22 张大阿尔卡纳 + 56 张小阿尔卡纳）承载人生旅程的原型能量，每张牌都有**正位**与**逆位**两种含义。二者天然契合——星盘定人，塔罗照事。

**需要问卜人提供的信息**（精度递增）：

| 信息 | 用途 | 必需度 |
|------|------|--------|
| 出生**日期** | 定太阳星座（核心自我） | 必需 |
| 出生**时间** | 定上升星座与宫位 | 建议 |
| 出生**地点** | 结合时间换算地方恒星时，定上升点 | 建议 |
| 当前**问题/领域** | 聚焦解读（爱情/事业/财富…） | 建议 |

本应用把占星与塔罗融合：**用本命盘信息作为种子决定抽到哪些牌、正位还是逆位**，同一人、同一问题、同一天结果稳定（盘由人定），再用翻牌动画呈现。

## 功能

- 录入出生日期(必填)/时间/地点/所问领域，计算太阳星座本命盘
- **上升星座估算**：填写出生时间＋地点后，由儒略日→格林尼治恒星时→地方恒星时→黄赤交角与纬度推算上升点，映射到黄道星座（内置数十座中外城市经纬度表，未识别则用默认坐标）
- 完整 **78 张塔罗牌**（22 大阿尔卡纳 + 56 小阿尔卡纳，含权杖/圣杯/宝剑/星币四花色）
- 确定性抽牌（mulberry32 伪随机 + 出生信息哈希种子），并决定正/逆位
- 每张牌含正位/逆位释义、关键词、占星（元素/行星）对应与建议
- 4 套三张牌阵：综合运势 / 感情姻缘 / 事业学业 / 财富机遇
- 塔罗牌 3D 翻转动画逐张揭示，逆位牌图案倒转并标注「逆位」
- 结合太阳/上升星座与正逆位整体倾向生成综合寄语
- 移动端适配，深夜星空 + 烫金主题

## 本地预览

任选一种：

```bash
# 方式一：直接双击 index.html 用浏览器打开

# 方式二：起本地服务器（推荐，避免个别浏览器 file:// 限制）
npx http-server -p 8080
# 或
python3 -m http.server 8080
# 浏览器打开 http://localhost:8080
```

## 部署到 GitHub Pages

1. 把本文件夹内容上传到你的仓库（可放在根目录，或 `docs/` 目录）
2. 仓库 Settings → Pages → Source 选择对应分支/目录
3. 访问 GitHub 提供的 Pages 链接即可
